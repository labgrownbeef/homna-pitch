# 4.3 API Documentation

**4.3.1 Authentication**

```
POST /api/v1/auth/token
```

Request:

```
{
  "api_key": "your_api_key"
}
```

Response:

```
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "bearer",
  "expires_in": 3600
}
```
