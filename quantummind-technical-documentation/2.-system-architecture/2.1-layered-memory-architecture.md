# 2.1 Layered Memory Architecture

| Memory Layer      | Function                                                  | Retention Period | Processing Priority |
| ----------------- | --------------------------------------------------------- | ---------------- | ------------------- |
| Working Memory    | Immediate market signals and price action                 | 1-5 days         | Highest             |
| Short-Term Memory | Recent market events and sentiment shifts                 | 5-30 days        | High                |
| Long-Term Memory  | Historical patterns, market regime characterization       | 30-365 days      | Medium              |
| Permanent Memory  | Core market knowledge, economic principles, trading rules | Permanent        | Reference           |

Each memory layer employs sophisticated vector embeddings to store and retrieve relevant information based on contextual similarity.
