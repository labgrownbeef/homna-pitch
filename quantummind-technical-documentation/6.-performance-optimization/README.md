# 6. Performance Optimization

Memory parameters significantly impact system performance. The following table provides recommended settings for different market types:

| Parameter               | Global Markets | Saudi Market | Volatile Markets |
| ----------------------- | -------------- | ------------ | ---------------- |
| Working Memory Span     | 3 days         | 2 days       | 1 day            |
| Short-Term Memory Span  | 21 days        | 14 days      | 7 days           |
| Long-Term Memory Weight | 0.3            | 0.25         | 0.15             |
| Recency Bias Factor     | 0.7            | 0.8          | 0.9              |
