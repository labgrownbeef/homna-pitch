# 7.4 Logging and Audit

* **Log Types:**
  * System logs: OS and infrastructure events
  * Application logs: Trading system activity
  * Trading logs: Order and execution details
  * Security logs: Access and security events
* **Log Management:** Centralized collection, indexing, and analysis
* **Retention:** Tiered retention policy based on data criticality
* **Compliance:** Tamper-evident storage for regulatory purposes
