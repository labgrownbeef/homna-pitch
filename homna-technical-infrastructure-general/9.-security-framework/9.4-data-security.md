# 9.4 Data Security

* **Data Classification:** Tiered classification based on sensitivity
* **Encryption:** Data-at-rest encryption for sensitive information
* **Access Controls:** Need-to-know access restrictions
* **Data Loss Prevention:** Controls to prevent unauthorized data exfiltration
* **Secure Disposal:** Procedures for secure data and media destruction
