# 9.2 Network Security

* **Perimeter Defense:**
  * Next-generation firewalls
  * DDoS protection services
  * Traffic filtering and inspection
* **Network Segmentation:**
  * Separate trading, management, and backup networks
  * Controlled inter-segment communication
  * Micro-segmentation for critical systems
* **Encryption:**
  * TLS for external connections
  * VPN for remote access
  * Hardware-accelerated encryption for data links
* **Monitoring:**
  * Network behavior analysis
  * Intrusion detection/prevention systems
  * Network traffic recording and analysis
