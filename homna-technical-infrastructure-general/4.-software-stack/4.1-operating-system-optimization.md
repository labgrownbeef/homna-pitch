# 4.1 Operating System Optimization

**Linux Kernel Optimizations**

* **Base Distribution:** CentOS/RHEL 8.x with RT-PREEMPT patches
* **CPU Isolation:** isolcpus, nohz\_full, and rcu\_nocbs kernel parameters to dedicate cores
* **Memory Management:**
  * Huge pages (2MB and 1GB) for trading processes
  * Disabled transparent huge pages
  * Locked memory allocations (mlock)
* **Process Scheduling:** SCHED\_FIFO for critical trading threads
* **System Services:** Minimal service configuration, disabled unnecessary daemons
* **Power Management:** CPU frequency scaling disabled, C-states limited to C1
* **IRQ Affinity:** Network interrupts assigned to specific non-trading cores
