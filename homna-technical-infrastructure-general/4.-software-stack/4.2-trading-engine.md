# 4.2 Trading Engine

* **Primary Implementation Language:** C++ (C++20 standard)
* **Critical Paths:** Custom memory allocators, lock-free data structures
* **Compiler Optimization:** Profile-guided optimization, link-time optimization
* **Performance Targets:**
  * Market data processing: <10 μs
  * Signal generation: <20 μs
  * Order decision: <10 μs
  * Order submission: <10 μs
* **Architecture:** Event-driven, multi-threaded with precise thread affinity

**Code Example: Low-Latency Memory Allocator**

```cpp
// Custom pool-based allocator for order objects
template<typename T, size_t BlockSize = 4096>
class PoolAllocator {
private:
    struct Block {
        char data[BlockSize];
        Block* next;
    };
    
    class FreeList {
    private:
        struct FreeNode {
            FreeNode* next;
        };
        
        FreeNode* head = nullptr;
        
    public:
        void push(void* ptr) noexcept {
            auto node = static_cast<FreeNode*>(ptr);
            node->next = head;
            head = node;
        }
        
        void* pop() noexcept {
            if (!head) return nullptr;
            auto node = head;
            head = head->next;
            return node;
        }
    };
    
    Block* currentBlock = nullptr;
    std::size_t currentSlot = 0;
    FreeList freeList;
    
public:
    T* allocate() {
        if (void* ptr = freeList.pop()) {
            return static_cast<T*>(ptr);
        }
        
        if (!currentBlock || currentSlot >= BlockSize / sizeof(T)) {
            // Allocate new block
            auto newBlock = new Block;
            newBlock->next = currentBlock;
            currentBlock = newBlock;
            currentSlot = 0;
        }
        
        T* result = reinterpret_cast<T*>(currentBlock->data + currentSlot * sizeof(T));
        currentSlot++;
        return result;
    }
    
    void deallocate(T* ptr) noexcept {
        if (ptr) freeList.push(ptr);
    }
};
```
