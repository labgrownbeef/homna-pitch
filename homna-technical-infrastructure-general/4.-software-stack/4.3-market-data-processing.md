# 4.3 Market Data Processing

* **Feed Handlers:**
  * Direct exchange protocol implementations (no middleware)
  * FPGA-accelerated for Tadawul exchange
  * Software-based handlers for secondary markets
* **Data Normalization:** Protocol-specific normalizers to convert to internal format
* **Distribution System:** Zero-copy shared memory transport between processes
* **Persistent Storage:** Compressed time-series database for historical data

\
Market data processing: <10 μs

* Signal generation: <20 μs
* &#x20;Order decision: <10 μs
* Order submission: <10 μs
