# 4.6 Order Execution

* **Execution Methods:**
  * Direct market access (DMA)
  * FIX protocol interfaces
  * Native exchange protocols
  * REST APIs for non-latency-sensitive venues
* **Order Types Supported:** Limit, Market, Stop, Stop-Limit, Stop-Loss, Iceberg, Pegged, TWAP/VWAP, Time in Force / Good-Till-Canceled (\[1] GTC), Immediate-Or-Cancel (IOC), Fill-Or-Kill (FOK)
* **Smart Order Routing:** Dynamic venue selection based on market conditions
