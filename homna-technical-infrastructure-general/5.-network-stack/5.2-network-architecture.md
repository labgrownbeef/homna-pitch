# 5.2 Network Architecture

* **Trading Network:** Dedicated, isolated network for market data and order flow
* **Management Network:** Separate network for administration and monitoring
* **Backup Network:** Independent path for disaster recovery
* **Network Equipment:** Arista 7130 Series ultra-low-latency switches
* **Topology:** Star configuration with minimal hop count to exchange points
