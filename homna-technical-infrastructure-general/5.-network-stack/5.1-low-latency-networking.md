# 5.1 Low-Latency Networking

**Network Optimization Techniques**

* **Kernel Bypass:** Direct hardware access using DPDK or Solarflare TCPDirect
* **TCP Offloading:** Hardware-based TCP processing on specialized NICs
* **RDMA:** Remote Direct Memory Access for inter-server communication
* **NIC Configuration:**
  * Interrupt coalescing disabled
  * Receive Side Scaling (RSS) optimized
  * Flow steering to dedicated CPU cores
* **Socket Buffers:** Optimized buffer sizes for trading traffic patterns
