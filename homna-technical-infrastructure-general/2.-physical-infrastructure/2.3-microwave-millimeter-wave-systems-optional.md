# 2.3 Microwave/Millimeter Wave Systems (Optional)

* **Spectrum:** Licensed 18GHz, 23GHz, and 80GHz bands
* **Key Routes:**
  * Riyadh to Jeddah: Primary and redundant paths
  * Riyadh to Dammam: Primary path
* **Availability:** 99.95% with automatic failover to fiber
* **Latency Advantage:** 30-40% improvement over fiber connections
* **Weather Mitigation:** Adaptive modulation and path diversity
