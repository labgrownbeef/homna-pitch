# 3.3 AWS Architecture

**AWS Services Utilized**

* **Compute:** EC2 instances (c6i and r6i families) for analytics workloads
* **Storage:** S3 for market data archives, EBS for operational data
* **Database:** DynamoDB for tick data storage, RDS for relational data
* **Analytics:** EMR for batch processing, Kinesis for streaming analytics
* **Security:**
  * KMS for encryption key management
  * AWS Shield for DDoS protection
  * CloudTrail for audit logging
  * VPC endpoints for private service access
* **Networking:** Direct Connect (10Gbps) to colocation facilities, Transit Gateway for cross-region connectivity
