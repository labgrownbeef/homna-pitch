# 3.1 Custom Trading Servers

**Trading Server Specifications**

* **Processors:** Intel Xeon Platinum 8380 (Ice Lake) with selected cores isolated for trading processes
* **CPU Configuration:** Frequency-optimized at 3.4GHz with Advanced Vector Extensions (AVX-512)
* **RAM:** 256GB DDR4-3200 ECC registered memory with optimized NUMA configuration
* **Storage:** NVMe SSDs in RAID 1 for OS/application and separate NVMe drives for market data persistence
* **Network:** Dual-port 25GbE network adapters with hardware timestamping
* **OS:** Custom-tuned Linux kernel (CentOS/RHEL based) with RT-PREEMPT patches
