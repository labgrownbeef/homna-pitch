# 5.1 Network Configuration

**Multicast Setup**

```bash
#!/bin/bash
# Configure multicast for ITCH Total View

# Enable multicast
ip link set dev ens1f0 multicast on

# Join multicast groups
ip maddr add 239.192.10.1 dev ens1f0  # ITCH-A
ip maddr add 239.192.10.2 dev ens1f0  # ITCH-B

# IGMP configuration
echo 2 > /proc/sys/net/ipv4/conf/ens1f0/force_igmp_version

# Increase receive buffer
sysctl -w net.core.rmem_max=134217728
sysctl -w net.core.rmem_default=67108864
```

**TCP Optimization for SoupBinTCP**

```bash
# TCP settings for ITCH News and GLIMPSE
sysctl -w net.ipv4.tcp_low_latency=1
sysctl -w net.ipv4.tcp_sack=0
sysctl -w net.ipv4.tcp_timestamps=0
sysctl -w net.core.netdev_max_backlog=5000
sysctl -w net.ipv4.tcp_no_delay_ack=1
```
